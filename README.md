## Usage

### Size Change
`/size <size>` for non-op player  
`/size <size> [players]` for op player  

### Whitelist
`/gulliver whitelist <on|off>`  
`/gulliver whitelist <add|remove> <players>`  

### Encoding

Files must be encoded as UTF-8.
